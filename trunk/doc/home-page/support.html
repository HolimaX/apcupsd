Support

Please keep in mind that we are not getting paid for this. Nevertheless, our desire is to see as many people using apcupsd as possible.

As with bug reporting, there two ways of asking for support.

The simplest way is to send an email to the apcupsd-devel@apcupsd.com email list. Please see Information Needed below for what to include in your support request.

The second way is to use the support tracking tool on SourceForge at http://sourceforge.net/tracker/?group_id=54413&func=browse.
This second method is less convenient for the the developers as it requires us to login to SourceForge to respond. However, this method is preferrable if you are submitting a bug or if you want your request tracked. You do not need to have a login to SourceForge to use either the mailing list or any of the reporting/tracking tools.

Which ever method you use, please follow the instructions below.

Information Needed

For us to respond to a support request, we normally need the following as the minimum information:
  • Your operating system
  • The version of apcupsd you are using
  • The model and UPS cable you are using
  • A clear description of the problem
The first three of these items can be fulfilled by sending us a copy of your config.out file, which is in the main apcupsd source directory after you have done your ./configure.

In addition, we will often need a copy of your apcupsd configuration file. If you think it is a configuration problem, please don't hesitate to send them preferably as an attachment.
